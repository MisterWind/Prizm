Чтобы создать обычный идикатор, создайте элемент и добавьте ему класс zui-indicator

Чтобы изменить состояние, добавьте элементу с классом zui-indicator название одного из состояний default | info | success | warning | danger

Чтобы индикатор был с иконкой, добавьте элементу с классом zui-indicator класс icon
