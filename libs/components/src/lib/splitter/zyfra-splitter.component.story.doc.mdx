## Zyfra - ZyfraSplitter

<Subtitle of={ZyfraSplitterComponent}>Splitter is utilized to separate and resize panels.</Subtitle>

Import module:

```typescript
import { ZyfraSplitterModule } from '@digital-plant/zyfra-components';

@NgModule({
  imports: [ZyfraSplitterModule],
})
export class YourModule {}
```

To component:

```html
<zyfra-splitter [panelSizes]="[50, 0]" [minSizes]="[1, 0]" [style]="{ height: '300px' }">
  <ng-template zyfraSplitterTemplate [visible]="false">
    <div>Panel 1</div>
  </ng-template>
  <ng-template zyfraSplitterTemplate>
    <div>Panel 2</div>
  </ng-template>
</zyfra-splitter>
```
